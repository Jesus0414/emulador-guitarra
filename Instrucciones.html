Descripción Nuestro proyecto consiste en un sitio web que se conforma de 3 páginas, es acerca de una guitarra eléctrica que se puede tocar por medio del teclado, cada tecla reproduce un sonido distinto, unas reproducen el sonido de las cuerdas, otras la de los trastes y si se combina una tecla de cuerda con alguna de los trastes se reproducirá el sonido de su combinación, así como sucede en una guitarra eléctrica física. ¿En que nos basamos? Este proyecto se realizo en base a un ejercicio realizado en clase que consiste también en un sitio web conformado por solamente una pagina en la cual es acerca de un piano que se puede tocar con las teclas del teclado, hay distintos botones para cada una de las teclas del piano. Nuestro proyecto lo realizamos de igual manera de un instrumento musical, pero lo desarrollamos de una forma aun mas compleja con la combinación de teclas y colocar aún más sonidos. ¿Cómo funciona? Para tocar las cuerdas se tocan del 1 al 6 y para los trastes son las teclas “q”, “w”, “e”, “r”, “t”, “y”, “u”, “i”, “o”, “p”, “{” y “}” ¿Cómo lo hicimos? Para realizar este proyecto creamos una carpeta con el nombre del proyecto y dentro de ella, 5 archivos por medio de la consola de PowerShell cada uno con los nombres y extensiones: index.html, app.js, style.css. El primero se llama index porque hace referencia a la pagina principal del sitio web y ese nombre esta reservado para que los servidores detecten esa pagina como la principal del sitio web y es un html porque todo sitio web necesita de un lenguaje de hipertexto, es básicamente, lo que le da el cuerpo al sitio, además de este html creamos el html del instructivo de una segunda guitarra, el cuarto archivo consiste en un archivo de JavaScript el cual es un lenguaje de programación que hace que el sitio web sea funcional y realice todo lo que tenga que hacer y por ultimo el quinto archivo es una hoja de estilo en cascada el cual es un lenguaje de diseño grafico que ayuda a crear el diseño del sitio y de cada uno de sus elementos. Una vez creados estos archivos desde la consola de PowerShell se abrió la aplicación de Visual Studio Code en donde por medio de Git se subió el proyecto en un repositorio en Github para poder darle cambios al código. En el html le colocamos todas las etiquetas necesarias, como la de los sonidos, la imagen de la guitarra etc. En el js se programó todas las funciones del sitio como la reproducción de sonido por medio de una tecla o su combinación, y en el css el color del fondo o de los botones. Y en PhotoShop editamos la imagen de la Guitarra.
